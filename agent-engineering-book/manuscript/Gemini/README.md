# Agent Engineering with Gemini, ADK & Antigravity

This is the Gemini/Google Cloud edition of the Agent Engineering series — the same methodology, case study, and Seven Steps framework as the Claude edition, implemented with Gemini, Google's Agent Development Kit (ADK), and the Antigravity development environment.

- **[`book-1-Foundations/`](book-1-Foundations/)** — Book 1: build the bounded WidgetWare SDR Lab application, chapter by chapter, through Chapter 11's engineered loop. Start with `book-1-Foundations/00_README.md`.
- **[`book-2-Advanced-Architectures/`](book-2-Advanced-Architectures/)** — Book 2: extend that application into an enterprise agent platform — memory, enterprise retrieval, adaptive planning, distributed agent collaboration, identity, governance, AgentOps, and continuous evaluation. Start with `book-2-Advanced-Architectures/00_README.md`.
- **[`course/`](course/)** — the instructor-facing companion: both books delivered as a ten-class, cumulative program with a fixed classroom cadence, golden-solution checkpoints, and a three-level homework pattern. Start with `course/00_Course_Framework.md`.

Both books use the same seven-step methodology as every edition of this series — Frame the Use Case, Build Context, Design Agent Capabilities, Build the Harness, Orchestrate Workflows, Engineer Loops, Evaluate & Govern — and the same continuous case study, WidgetWare SDR Lab, with identical business facts (product, ideal customer profile, evidence policy) across both books. Only the implementation technology differs from the Claude edition.
